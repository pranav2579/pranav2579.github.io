# Pranav Tripathi — Portfolio

🌐 **Live:** [https://pranav2579.github.io](https://pranav2579.github.io)

A personal portfolio website built with [Next.js](https://nextjs.org) and deployed on GitHub Pages.

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view locally.

## Tech Stack

- [Next.js](https://nextjs.org) (App Router)
- [TypeScript](https://www.typescriptlang.org)
- [Geist Font](https://vercel.com/font)

## Deployment

This site is deployed via GitHub Pages at [pranav2579.github.io](https://pranav2579.github.io).
